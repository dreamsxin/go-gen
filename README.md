

## 安装使用

```shell
go install github.com/dreamsxin/go-gen@latest
mkdir test && cd test
go mod init hello
go-gen init
```

## 初始化

```shell
go-gen init
```

## 创建 init 命令

```shell
go-gen add init
```

## 创建 gen 命令

```shell
go-gen add gen
```

## 创建 gen 命令

```shell
go-gen add gen
```
    
`
go-gen init
```

## cobra cli

### 初始化

```shell
cobra-cli init --viper --author "dreamsxin@qq.com" --license "Apache-2.0"
```

### 创建 init 命令

```shell
cobra-cli add init
```
